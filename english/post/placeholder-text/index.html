Khái Niệm Tính Trừu Tượng Trong Java

Tính trừu tượng là một trong bốn nguyên lý cơ bản của lập trình hướng đối tượng (OOP) trong Java, cùng với đóng gói, kế thừa và đa hình. Nó cho phép lập trình viên tập trung vào các khía cạnh quan trọng của đối tượng trong khi ẩn đi các chi tiết triển khai không cần thiết. Tính trừu tượng giúp giảm bớt sự phức tạp và tăng hiệu quả trong việc phát triển phần mềm.

Trong Java, tính trừu tượng được thực hiện thông qua việc sử dụng lớp trừu tượng và giao diện.

1. Lớp Trừu Tượng

Lớp trừu tượng trong Java là một lớp không thể được khởi tạo trực tiếp. Nó hoạt động như một bản thiết kế cho các lớp khác. Lớp trừu tượng có thể chứa cả phương thức trừu tượng (phương thức không có triển khai) và phương thức cụ thể (phương thức có triển khai).

abstract class Animal {
    abstract void sound();
    
    void sleep() {
        System.out.println("Con vật này đang ngủ.");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("Gâu gâu");
    }
}
